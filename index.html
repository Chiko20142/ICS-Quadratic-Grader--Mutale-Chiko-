# ICS Quadratic Grader ## What the page does -Let's you input values for a, b, and c. -Calculates the discriminant (b^2 - 4ac). -Displays the real or complex roots on the webpage. ## How to Run it 1. Download or clone this repository. 2. Open the 'index.htm' file in any web browser. 3. Enter values for a, b, and c, then click **Solve**. 4. ##Example 5. For a = 1, b = -3, c = 2 6. Roots are x = 2 and x = 1
